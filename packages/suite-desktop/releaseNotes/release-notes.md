### 🚀 New features

-   Users can now cancel outgoing Bitcoin transactions using Replace-By-Fee (RBF), providing more flexibility and control over their transactions.
-   Stake Solana (SOL) directly from Trezor Suite. Easily receive rewards by delegating your SOL through the new “Stake” feature in the assets and trade table.
-   A new Over-the-Counter (OTC) trade flow is now available for high-value Buy and Sell transactions that cannot be fulfilled through standard trading options.

### 🎨 Improvements

-   A new receive confirmation modal guides users through the process of safely verifying addresses end to end to enhance security.
-   The redesigned welcome screen provides more space for important content, improves alignment with the in-app menu bar, and removes unnecessary empty space.
-   Increased security of application files on Windows by improving integrity protection.
-   Solana transactions now display a Blockhash timer, giving users better visibility into transaction expiration and timing.
-   Integrated 1inch Fusion for significantly cheaper DEX swap transactions, helping you save on trading fees.

### 🔧 Bug fixes

-   Fixed minor bugs, improved usability, and optimized performance for a smoother experience.
