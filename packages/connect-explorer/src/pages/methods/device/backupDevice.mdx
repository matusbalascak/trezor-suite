## Backup device

<Callout type="error">
    **Management command** - this method is restricted to Trezor.io and can't be used in 3rd party
    applications.
</Callout>

Start seed backup process on the device. Can only be called if the seed is not backed up yet.

```javascript
const result = await TrezorConnect.backupDevice(params);
```

### Params

<CommonParamsLink />

<ParamsTable schema={PROTO.BackupDevice} />

### Example

```javascript
TrezorConnect.backupDevice({
    device,
});
```

### Result

Success type

```javascript
{
    success: true,
    payload: {
        message: 'Seed successfully backed up'
    }
}
```

Error

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
